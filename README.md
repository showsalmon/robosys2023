# robosys2023
ロボットシステム学の練習リポジトリ

# plusコマンド
[![test](https://github.com/showsalmon/robosys202x/actions/workflows/test.yml/badge.svg)](https://github.com/showsalmon/robosys202x/actions/workflows/test.yml)
このソフトは、標準入力から読み込んだ数字kまでを
Σk, Σk^2, Σk^3, k!
の計算を行います.

## 必要なソフトウェア
* Python
  * テスト済み: 3.7～3.10

## テスト環境
* Ubuntu 20.04
